# FORM BUILDER - Drag-and-Drop Form Builder

FORM BUILDER is a modern, intuitive form builder application built with React, TypeScript, and Vite. It allows users to create custom forms by dragging and dropping fields from a palette onto a canvas, configuring field settings, and previewing or publishing the forms.

⚠️ **Note**: This project is currently in development. Features may be incomplete, and there could be bugs or breaking changes as development progresses.

## Features

- **Drag-and-Drop Interface**: Easily drag form fields from the palette to the canvas using @dnd-kit.
- **Field Types**: Support for various field types including text inputs, selects, checkboxes, and more.
- **Field Settings**: Customize field properties such as labels, placeholders, required status, and validation.
- **Responsive Design**: Built with Tailwind CSS for a clean, responsive UI.
- **Preview and Publish**: Preview your form before publishing it.
- **TypeScript Support**: Fully typed with TypeScript for better development experience.

## Tech Stack

- **Frontend**: React 19, TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **UI Components**: Radix UI Themes
- **Drag-and-Drop**: @dnd-kit
- **Form Handling**: React Hook Form with Zod validation
- **Icons**: Lucide React

## Getting Started

### Prerequisites

- Node.js (version 18 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/aysenurozbay/form-builder.git
   cd form-builder
   ```
2. Install dependencies:
   ```bash
   npm install
   # or
   yarn install
   ```
3. Start the development server:
   ```bash
   npm run dev
   # or
   yarn dev
   ```

Your browser should automatically open `http://localhost:5173`. If not, you can manually open it.

## Usage

- Open the application in your browser.
- Start by dragging form fields from the right-hand palette onto the canvas.
- Configure field settings by clicking on the fields in the canvas.
- Use the preview feature to see how your form will look.
- Publish your form to make it live.
